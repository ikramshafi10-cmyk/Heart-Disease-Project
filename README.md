## ❤️ Heart Disease Detection 
📌 Project Overview

This project focuses on analyzing and predicting Heart Disease using data science / machine learning techniques. The goal is to help in early diagnosis and reduce health risks.

## 🎯 OBJECTIVES

Identify key factors that contribute to heart disease

Build a predictive model for early detection

Improve accuracy using different algorithms

Provide insights for better healthcare decisions

## 📊 DATASET

Features may include:

1.Age
2.Sex
3.RestingBP
4.ChestPainType
5.Cholesterol
6.FastingBS
7.RestingECG
8.MaxHR
9.ExerciseAngina
10.Oldpeak
11.ST_Slope
12.HeartDisease

Target: heart disease / not heart disease


## ⚙️ Technologies Used

Python

Libraries: Pandas, NumPy, Matplotlib, Scikit-learn

Jupyter Notebook / VS Code


## 🧠 Machine Learning Models

1.Logistic Regression
2.Decision Tree
3.GaussianNB
4.Support Vector Machine (SVM)
5.KNN

📈 Results

Accuracy: 88%
Best Model: (SVM)
Observations:
High cholesterol and age are major factors.
